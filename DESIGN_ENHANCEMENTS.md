## 🎬 Animation Types Used

| Animation | Duration | Easing | Purpose |
|-----------|----------|--------|---------|
| fadeInUp | 0.6s | ease-out | Elements entering view |
| fadeIn | 0.8s | ease-out | Opacity transitions |
| slideInLeft/Right | 0.6s | ease-out | Side entry animations |
| scaleIn | 0.5s | ease-out | Growth animations |
| float | 20s | ease-in-out | Floating effects |
| pulse | 2s | ease-in-out | Breathing animations |
| shimmer | 1.5s | ease | Shine effects |
| gradient-shift | 15s | ease | Background gradients |
| glow | 2s | ease-in-out | Glowing effects |
| spin | 1s | linear | Loading spinner |
| ripple | 0.6s | ease-out | Click feedback |
